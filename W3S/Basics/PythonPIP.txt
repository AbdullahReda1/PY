==========================================
Python PIP (Package Manager)
==========================================


------------------------------------------------
What is PIP?
PIP is a package manager used to install and manage
Python packages from the Python Package Index (PyPI)
------------------------------------------------


------------------------------------------------
Check if PIP is Installed
------------------------------------------------
pip --version


------------------------------------------------
Install PIP (if not installed)
------------------------------------------------
python -m ensurepip --upgrade


------------------------------------------------
Install a Package
------------------------------------------------
pip install package_name

Example:
pip install numpy


------------------------------------------------
Install Specific Version
------------------------------------------------
pip install package_name==version

Example:
pip install numpy==1.24.0


------------------------------------------------
Upgrade a Package
------------------------------------------------
pip install --upgrade package_name


------------------------------------------------
Uninstall a Package
------------------------------------------------
pip uninstall package_name


------------------------------------------------
List Installed Packages
------------------------------------------------
pip list


------------------------------------------------
Show Package Information
------------------------------------------------
pip show package_name


------------------------------------------------
Freeze Installed Packages
Creates a requirements file
------------------------------------------------
pip freeze > requirements.txt


------------------------------------------------
Install from requirements.txt
------------------------------------------------
pip install -r requirements.txt


------------------------------------------------
Search for Packages (Deprecated in newer pip)
------------------------------------------------
pip search package_name


------------------------------------------------
Check Outdated Packages
------------------------------------------------
pip list --outdated


------------------------------------------------
Upgrade All Packages (manual loop idea)
------------------------------------------------
pip list --outdated
pip install --upgrade package_name


------------------------------------------------
User-level Installation (no admin rights)
------------------------------------------------
pip install --user package_name


------------------------------------------------
Install from URL or Git
------------------------------------------------
pip install https://example.com/package.zip
pip install git+https://github.com/user/repo.git


------------------------------------------------
Uninstall Multiple Packages
------------------------------------------------
pip uninstall package1 package2


------------------------------------------------
Cache Management
------------------------------------------------
pip cache dir
pip cache purge


------------------------------------------------
Best Practices
------------------------------------------------
- Use virtual environments (venv)
- Keep requirements.txt updated
- Avoid global installs when possible
- Pin versions for reproducibility
